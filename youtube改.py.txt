Step.1

https://www.python.org/downloads/

このページから最新版のpython3.8.0をダウンロードできます。

ダウンロードしたらインストールを開始できます。

ダブルクリックです。

そうしたら画面にウィザードが立ち上がります。

僕はAdd Python 3.8 to PATHにチェック入れました。

Install Now でインストールが始まります。

インストールする場合は　はい　を選択してください。

Step.2

cmd を立ち上げます。

windows、スタートメニューから

検索ボックスにコマンドプロンプトと入力して [コマンドプロンプト] をクリックします。

立ち上げたら C:\Users\ここはそれぞれのユーザー名> とゆう画面になっていると思います。

仮にユーザー名をtestuserだとしたら、ダウンロードしたyoutube改.pyをC:\Users\testuserのフォルダに配置してください。

pip install html5lib
pip install BeautifulSoup4
pip install requests
pip install youtube_dl

を実行してくださいね。

後は画面に python youtube改.py キーワード を入力してください。

キーワードはそれぞれ任意の文字列です。
